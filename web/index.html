Описание

Keyhole bot

Keyhole bot позволяет транслировать переписку из групп в каналы.

Дает возможность людям подглядывать за разговорами, которые ведет закрытая группа людей.

Настраивается трансляция в несколько простых шагов:
  1. Добавляем бота в канал
  2. Добавляем бота в группу
  3. Связываем канал и группу

Бот бесплатен. Приятного использования

Оглавление:

  1. Подключение бота в группу
  2. Подключение бота в канал

Описание команд

/getChannelCode

Используется только в групповом чате

Бот предоставляет код, который используется для связывания группы и канала

/useCode

Используется только в личном чате с ботом.

Эта команда связывает канал и групповой чат

/addChannel

Используется только в личном чате с ботом.

Эта команда сообщает боту что его добавили в канал. Только после этого бот сможет связать группу и публиковать сообщения
